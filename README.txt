A project for the Information Retrieval(IR) class at Jordan University of Science and Technology - Fall 2013/2014

The aim of the project:
1- Build a documents pre-processor that does several techniques to ease the job of IR systems.
2- Build an IR system based on the vector distance model that allows a user to query the system over a chosen set of 
	documents.

Style notes:
	We're following Microsoft's .NET style guidlines as closly as possible.
